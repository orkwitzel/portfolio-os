# Keyboard shortcuts

| Chord | Action |
|-------|--------|
| **Escape** | Close the focused window (with unsaved prompt in Notepad when applicable) |
| **Alt+F4** | Close the focused window (same guard as Escape) |
| **Ctrl+`** | Cycle focus through visible windows |
| **Ctrl+S** | Save in Notepad |
| **Ctrl+N / Ctrl+O** | New / Open in Notepad |
| **Ctrl+F / F3** | Find / Find Next in Notepad |

## Notes

- When the Start menu is open, Escape closes the menu first.
- Alt+Tab is reserved by the OS; the shell uses Ctrl+` instead.
- Escape does not close a window while focus is in an editable field (blur the Notepad field first).
